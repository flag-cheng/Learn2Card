# Python 基礎

Python 是一種高階程式語言，易於學習且功能強大。

## 變數與資料型別

Python 支援多種資料型別，包括整數、浮點數和字串。

- 整數：用於表示整數值，如 1, 2, 3
- 浮點數：用於表示小數，如 3.14, 2.5
- 字串：用於表示文字，如 "Hello World"

## 函式定義

函式是可重複使用的程式碼區塊。

使用 def 關鍵字定義函式，可以接收參數並回傳值。

## 控制流程

Python 提供 if-else 條件判斷和迴圈結構。

- if-else：根據條件執行不同的程式碼
- for 迴圈：遍歷序列中的每個元素
- while 迴圈：當條件為真時重複執行
